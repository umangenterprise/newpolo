Frontend Me Kya Use Hua Hai

  • React 19 single-page application
  • Vite for fast development and production build
  • React Router DOM for page routing and protected routes
  • Axios for API communication
  • Framer Motion for animations and motion effects
  • React Hot Toast for success and error notifications
  • React Icons for UI icons
  • Context API for user, cart and product state

Backend Me Kya Use Hua Hai

  • Node.js runtime with Express.js API server
  • MongoDB database with Mongoose models
  • JWT token based authentication
  • bcryptjs for password hashing
  • Multer for image and payment screenshot uploads
  • Nodemailer for email sending
  • Razorpay SDK for online payments
  • CORS, dotenv and centralized error handling
